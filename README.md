# Tela de Login

Neste projeto, o objetivo é criar um cenário de teste documentado utilizando o JUnit e seguir o padrão TDD (Test Driven Development). Além disso, será gerado um Javadoc para documentar o código.

## Classe de Teste Login
Em conformidade com o padrão TDD, começamos criando os testes antes de desenvolver as classes do projeto. Isso nos ajuda a identificar os recursos necessários para o funcionamento do programa.

## Classe LoginScreen
Após identificar os erros nos testes, desenvolvemos a classe LoginScreen, incluindo os métodos necessários para a execução dos testes.
